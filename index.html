網頁顯示或資料更新異常？

若您遇到地震報告資訊 (如報告編號、地震規模、各地震度) 顯示為 N/A、網頁排版錯亂或資料未即時更新等問題，建議您採取以下步驟：

  1. **清除本地歷史記錄**：這是最重要的一步！請點擊下方的「清除本地歷史記錄」按鈕，這將清除瀏覽器中可能已經儲存的舊有或格式不符的地震資料。
  2. 重新整理網頁：清除歷史記錄後，請重新載入此網頁。這將確保載入最新的網頁程式碼，並開始獲取新的、正確格式的地震資料。
  3. 檢查網路連線：確保您的裝置網路穩定且暢通。
  4. 查看開發人員工具：按下 `F12` (或 Mac 上的 `Option + Command + I`) 開啟瀏覽器的開發人員工具，點擊 `Console` (控制台) 頁籤，查看是否有錯誤訊息；也可以點擊 `Network` (網路) 頁籤，檢查 API 請求的狀態和回傳的原始資料。
  5. API 服務狀態：偶爾中央氣象署的 API 服務可能會有短暫的異常或維護，導致資料暫時無法取得。
  6. 等待最新報告：對於較小的地震或初步報告，各地震度資訊可能不會立即提供，請耐心等待後續更新。

完成步驟 1 和 2 後，請觀察新的地震報告，並查看其「報告編號」和「各地震度」是否正常顯示。
